# Smart Wallet Engineering Journal

## Entry #3 — Beginning the CSS Journey: Building the Foundation of the Home Page

**Date:** 21 July 2026

### Today's Goal

Start designing the visual appearance of the Smart Wallet home page using CSS and move the project from a basic Flask-rendered page towards a proper user interface.

### Work Completed Today

- Started the CSS development for the Smart Wallet home page.
- Created the global styling foundation for the application.
- Added basic page styling:
  - Removed default browser margins and padding.
  - Selected the main font family for the application.
  - Added a soft background color and default text color.

- Styled the navigation header:
  - Used Flexbox for alignment.
  - Added spacing between elements.
  - Added a clean white header background.
  - Added a subtle shadow to improve the appearance.

- Created reusable button styles:
  - Designed a primary button for important actions.
  - Designed a secondary button with a border style.
  - Added padding, rounded corners, and text formatting.

- Added basic spacing and layout improvements for the main content section.

### Concepts Learned

- CSS selectors and how they target HTML elements.
- The importance of global styling before designing individual components.
- Using Flexbox (`display: flex`) to arrange elements.
- Understanding spacing using:
  - margin
  - padding
- Creating reusable CSS classes for buttons.
- Improving UI appearance using colors, shadows, and typography.

### Current Progress

The Smart Wallet home page now has:

- A basic visual identity.
- A consistent color theme.
- A styled header.
- Styled call-to-action buttons.
- A clean foundation for further UI development.

### Next Steps

- Improve the hero section of the home page.
- Add better typography hierarchy.
- Create cards/components for Smart Wallet features.
- Continue learning CSS layouts and responsive design.
- Gradually transform the page into a complete Smart Wallet interface.

### Reflection

Today was the first step in converting the Smart Wallet idea into an actual product interface. Instead of only focusing on functionality, I started building the user experience and visual identity of the application. The foundation created today will be expanded as more features are added.
